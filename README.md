# vue-pragram-test

> 基于vue搭建的自动聊天室和动态展示栏  
> vue全家桶(vuex,vue-router,axois) + node + express + mongodb 

## 安装与启动mongodb

* 安装

		$ brew install mongodb

* 启动MongoDb（安装成功后命令行有提示）
   
		$ mongod --config /usr/local/etc/mongod.conf

* 连接到mongo  

		$mongo
##启动服务端			
			npm run server
## 启动项目

``` bash
# 安装依赖
npm install

# 启动本地服务
npm run dev

# 生成生产版本
npm run build

# 单元测试
npm run unit

# e2e tests
npm run e2e

# run all tests
npm test
```
